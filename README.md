# Medical Hardy

Welcome to the **Medical Hardy** repository! This project is a comprehensive web-based platform designed for delivering high-quality medical courses. Our platform features a responsive design, offering users information on various courses, expert advice, and pricing options.

## Features

- **Home Section:** Introduction to Medical Hardy with quick navigation to other sections.
- **Products Section:** Displays popular courses such as Anatomy Essentials, Pathways to Mental Health, Pharmacology in Practice, and Emergency Medicine Mastery.
- **Pricing Section:** Details on pricing and subscription options.
- **Contact Section:** A form to subscribe to the newsletter and contact details.
- **Responsive Design:** Fully responsive web design ensuring a great experience on both desktop and mobile devices.

## Technologies

- HTML
- CSS
- FontAwesome for icons
- Google Fonts
## Usage
- Navigate to the website using the provided URL or local server address.
- Explore various courses by clicking on the 'Product' tab.
- Learn more about pricing under the 'Pricing' tab.
- Sign up or log in to access more personalized features.
- Contact support or get more information through the 'Contact' tab.
